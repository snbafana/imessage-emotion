# iMessage Emotion

Electron take-home app scaffold for visualizing emotion trends across a small
set of iMessage conversations.

## Architecture

- The Electron app lives at the repository root.
- `src/main.ts` owns desktop orchestration, OS access, IPC, and future Python
  subprocess calls.
- `src/preload.ts` is the safe bridge into the renderer.
- `src/renderer.ts` owns the UI entrypoint generated by the Electron Forge Vite
  template.
- `python/` is reserved for analysis and eval code only. It should not become a
  separate daemon or API server unless the app genuinely needs one. Forge copies
  it into packaged app resources for future subprocess execution.
- `fixtures/` is reserved for small local test/eval inputs.

## Development

```bash
npm install
npm start
```
